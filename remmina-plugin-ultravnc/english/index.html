Description

{{ site.data.remmina-plugin-ultravnc.name }} is a free software protocol plugin to establish connections using UltraVNC, an open source control VNC client with many features developed for MS Windows and executable under GNU/Linux via Wine.

UltraVNC installation is required in order to let this plugin to work properly.
