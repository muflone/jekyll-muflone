Descrizione

{{ site.data.remmina-plugin-ultravnc.name }} è un plugin di protocollo libero per stabilire connessioni usando UltraVNC, un'applicazione open source di controllo remoto VNC con numerose funzionalità sviluppata per MS Windows ed eseguibile in GNU/Linux attraverso Wine.

E' richiesta l'installazione di UltraVNC affinchè questo plugin funzioni.

Installazione

I rilasci delle versioni stabili con codice sorgente e pacchetti precompilati sono presenti nella pagina Scarica.

Il codice sorgente di {{ site.data.remmina-plugin-ultravnc.name }} è disponibile su GitHub con la versione attualmente in sviluppo.

Impostazioni

Le impostazioni del plugin {{ site.data.remmina-plugin-ultravnc.name }} sono documentate nella pagina Impostazioni.
