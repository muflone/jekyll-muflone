--- title: Introduzione ---

Descrizione

{{ site.data.remmina-plugin-open.name }} è un plugin di protocollo libero per aprire un file o un documento usando la sua applicazione associata dalla finestra di Remmina.

E' utile per avviare un'applicazione esterna associata ad un documento, come un gestore di password cifrate oppure un elenco con i sistemi in rete salvato in formato LibreOffice senza lasciare la finestra di Remmina.

Installazione

I rilasci delle versioni stabili con codice sorgente e pacchetti precompilati sono presenti nella pagina Scarica.

Il codice sorgente di {{ site.data.remmina-plugin-open.name }} è disponibile su GitHub con la versione attualmente in sviluppo.

Impostazioni

Le impostazioni del plugin {{ site.data.remmina-plugin-open.name }} sono documentate nella pagina Impostazioni.
