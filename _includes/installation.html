Installazione su Arch Linux e derivate

{% if include.development %}

{% when 'source' %}

Installazione dal codice sorgente

Se non fosse disponibile il pacchetto preassemblato per la propria distribuzione è possibile utilizzare il pacchetto col codice sorgente per installare {{ include.module.name }} utilizzando:
cd "~/cartella con il file scaricato"
mkdir {{ include.module.lowername }}-install
tar xvzf "nome del file.pkg.tar.gz" -C {{ include.module.lowername }}-install
cd {{ include.module.lowername }}-install/{{ include.module.lowername }}-*
python2 setup.py build
sudo python2 setup.py install

Al termine dell'installazione, se non sono stati riportati errori sarà possibile eliminare la cartella {{ include.module.lowername }}-install da root oppure tramite sudo rm -rf "{{ include.module.lowername }}-install" e il file sorgente scaricato.

Attenzione! Dopo aver eseguito la procedura di installazione manuale da codice sorgente non sarà possibile disinstallare automaticamente il software ma sarà necessario eliminare manualmente tutti i files installati che sono indicati durante l'ultimo passaggio del processo di installazione.

Installation on Debian, Ubuntu and derivatives

It's possible to download the deb package from the Download page and install it using:
cd "~/folder with the downloaded file"

{% when 'archlinux' %}

Installation on Arch Linux and derivatives

In the AUR you can find the latest available version, it can be downloaded and built using yaourt (or a similar AUR helper) using yaourt -S {{ include.module.lowername }} and then following the AUR helper instructions.

{% if include.development %}

The development version is also available in the AUR and can be installed using yaourt (or a similar AUR helper) with yaourt -S {{ include.module.lowername }}-git.

{% endif %}

For anyone which doesn't want to use AUR the tar.xz package can be downloaded from the Download page and installed using:
cd "~/folder with the downloaded file"
sudo pacman -U "file name.pkg.tar.xz"

{% when 'source' %}

Installation from source code

If the prebuilt package will not be available for your distribution you can use the source code package to install {{ include.module.name }} using:
cd "~/folder with the downloaded file"
mkdir {{ include.module.lowername }}-install
tar xvzf "file name.pkg.tar.gz" -C {{ include.module.lowername }}-install
cd {{ include.module.lowername }}-install/{{ include.module.lowername }}-*
python2 setup.py build
sudo python2 setup.py install

After the installation, if no errors were reported the folder {{ include.module.lowername }}-install can be removed from root or via sudo rm -rf "{{ include.module.lowername }}-install" and the same for the downloaded source package.

Warning! After executing the manual installation from source code an automatic software uninstall will not be possibile but you will have to manually delete all the installed files that were reported during the last installation step.
