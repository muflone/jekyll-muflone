Description

{{ site.data.remmina-plugin-exec.name }} is a free software protocol plugin to execute external processes (commands or applications) from the Remmina window.

Some examples of external applications could be third party remote access applications not supported by any Remmina protocol, network calculators, network diagram draw applications and many others.

Installation

The stable versions released with source code packages are linked in the Download page.

The {{ site.data.remmina-plugin-exec.name }} development version source code is available on GitHub.

Settings

The {{ site.data.remmina-plugin-exec.name }} plugin settings are documented in the Settings page.
