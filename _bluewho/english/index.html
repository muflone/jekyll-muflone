Description

{{ site.data.bluewho.name }} is a free application that will scan for Bluetooth devices near you will notify you when a new device is discovered.

When a new device is detected then a notification can be showed, accordingly to the preferences dialog.

Various preferences can ben set for scan and notifications.

For each detected device a list of the available services on the device can be obtained.

Installation

The stable versions released with source code and prebuilt packages are linked in the Download page. For information about installation from packages or source code please refer to the Installation page.

The {{ site.data.bluewho.name }} development version source code is available on GitHub.
