Description

{{ site.data.cpucompare.name }} is a free GTK+ application to make comparisons between CPU models. You can browse the CPU models by brands and series and find those of your insterest. Every CPU has a score based on benchmarks so you can determine what processor has greater or lower performances.

Installation

The stable versions released with source code and prebuilt packages are linked in the Download page. For information about installation from packages or source code please refer to the Installation page.

The {{ site.data.cpucompare.name }} development version source code is available on Github.
