Description

{{ site.data.remmina-plugin-webkit.name }} is a free software protocol plugin to open a web page using an integrated GTK+ WebKit web browser. It's useful when you need to access your resources from the Remmina window, like gaining access to routers administration web pages, printers and others resources in the network.

Installation

The stable versions released with source code packages are linked in the Download page.

The {{ site.data.remmina-plugin-webkit.name }} development version source code is available on GitHub.

Settings

The {{ site.data.remmina-plugin-webkit.name }} plugin settings are documented in the Settings page.
