Descrizione

{{ site.data.remmina-plugin-webkit.name }} è un plugin di protocollo libero per aprire una pagina web utilizzando un browser integrato basato su GTK+ WebKit. Diventa utile quando è necessario accedere alle risorse dalla finestra di Remmina, come avere accesso alle pagine web di amministrazione di router, stampanti e altre risorse in rete.
