Description

{{ site.data.remmina-plugin-rdesktop.name }} is a free software protocol plugin to establish a RDP connection using RDesktop which sometimes performs better than the FreeRDP integrated plugin in Remmina and also provides a lot of features like clipboard sync, remote folders support, remote audio, seamless RDP and various settings for performances.

RDesktop installation is required in order to let this plugin to work properly.
