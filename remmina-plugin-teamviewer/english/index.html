Description

{{ site.data.remmina-plugin-teamviewer.name }} is a free software protocol plugin to establish connections using TeamViewer, a remote control application with many features and capable of connecting even through NAT, both for Windows, Linux and others operating systems.

TeamViewer installation is required in order to let this plugin to work properly.

Installation

The stable versions released with source code packages are linked in the Download page.

The {{ site.data.remmina-plugin-teamviewer.name }} development version source code is available on GitHub.

Settings

The {{ site.data.remmina-plugin-teamviewer.name }} plugin settings are documented in the Settings page.
