Description

{{ site.data.remmina-plugin-folder.name }} is a free software protocol plugin to open a folder using an external file manager from the Remmina window.

It's useful to access to your documents folder or your settings without leaving the Remmina window.

Installation

The stable versions released with source code packages are linked in the Download page.

The {{ site.data.remmina-plugin-folder.name }} development version source code is available on GitHub.
