Descrizione

{{ site.data.remmina-plugin-folder.name }} è un plugin di protocollo libero per aprire una cartella utilizzando un gestore di files esterno dalla finestra di Remmina.

E' utile per accedere ad una cartella di documenti o di impostazioni senza lasciare la finestra di Remmina.

Installazione

I rilasci delle versioni stabili con codice sorgente e pacchetti precompilati sono presenti nella pagina Scarica.

Il codice sorgente di {{ site.data.remmina-plugin-folder.name }} è disponibile su GitHub con la versione attualmente in sviluppo.

Impostazioni

Le impostazioni del plugin {{ site.data.remmina-plugin-folder.name }} sono documentate nella pagina Impostazioni.
