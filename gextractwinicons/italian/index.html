Descrizione

{{ site.data.gextractwinicons.name }} è un'applicazione libera per estrarre cursori, icone e immagini PNG da file di risorse per Microsoft Windows (come .exe, .dll, .ocx e .cpl).

Installazione

I rilasci delle versioni stabili con codice sorgente e pacchetti precompilati sono presenti nella pagina Scarica. Per informazioni sull'installazione da pacchetti o con codice sorgente fare riferimento alla pagina Installazione.

Il codice sorgente di {{ site.data.gextractwinicons.name }} è disponibile su GitHub con la versione attualmente in sviluppo.
