Description

{{ site.data.gextractwinicons.name }} is a free application to extract cursors, icons and png images from Microsoft Windows resource files (like .exe, .dll, .ocx, .cpl).

Installation

The stable versions released with source code and prebuilt packages are linked in the Download page. For information about installation from packages or source code please refer to the Installation page.

The {{ site.data.gextractwinicons.name }} development version source code is available on GitHub.
