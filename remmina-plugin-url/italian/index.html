Descrizione

{{ site.data.remmina-plugin-url.name }} è un plugin di protocollo libero per aprire una pagina web utilizzando un browser externo. Diventa utile quando è necessario accedere alle risorse dalla finestra di Remmina, come avere accesso alle pagine web di amministrazione di router, stampanti e altre risorse in rete.

Installazione

I rilasci delle versioni stabili con codice sorgente e pacchetti precompilati sono presenti nella pagina Scarica.

Il codice sorgente di {{ site.data.remmina-plugin-url.name }} è disponibile su GitHub con la versione attualmente in sviluppo.
